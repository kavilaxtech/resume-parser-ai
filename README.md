# Resume Parser AI

A simple Flask web app that extracts Name, Email, Phone, and Skills from uploaded resumes (PDFs).

## Features
- Upload resume in PDF format
- Extracts:
  - Name
  - Email
  - Phone number
  - Skills (Python, Java, ML, etc.)
